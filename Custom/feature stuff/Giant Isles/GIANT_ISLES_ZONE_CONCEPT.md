**Zone Type:** Daily Quest Hub + World Bosses  
**Level Range:** 70-80 (Endgame Zone)  
**Ported From:** Isle of Giants (MoP 5.5.0)  
**Theme:** Prehistoric Chaos - Dinosaurs corrupted by Chaos energies  
**Map ID:** 1405 (Isle of Giants)

## ğŸŒ‹ Zone Overview

The **Giant Isles** is an ancient landmass where primordial beasts have survived since the dawn of Azeroth. Now, dark energies seep from dimensional rifts, corrupting these magnificent creatures into twisted weapons of destruction. The Zandalari trolls have already begun harnessing this power, breeding war-beasts to conquer the world.

Players must establish a foothold, push back the Zandalari invasion, and face three terrifying World Bosses that threaten to spread chaos across Azeroth.

## ğŸ­ DarkChaos Lore Integration

### The Corruption of the Isles

Long ago, the Titans sealed the Giant Isles away from the world, creating a sanctuary for the last primordial beasts. When the Dark Chaos began corrupting Azeroth, these seals weakened. Now:

- **Chaos Rifts** tear through the island, mutating creatures
- **The Zandalari** seek to control these enhanced beasts
- **Three Primordial Lords** have awakened, each touched by different aspects of Chaos
- **Both factions** scramble to claim the island's resources

### Faction: The Primal Wardens

A combined expedition of Horde and Alliance druids, hunters, and scholars who seek to:
- Study the primordial creatures
- Contain the Chaos corruption
- Prevent the Zandalari from weaponizing the beasts
- Harvest rare materials for powerful gear

### Zone-Wide Announcement System

World bosses use **map-wide text broadcasts** when spawning, at health thresholds, and on death:

```cpp
// Implementation example - uses Map::SendZoneText for zone announcements
void AnnounceWorldBoss(Creature* boss, const char* message)
{
    if (!boss || !boss->GetMap())
        return;
    
    // Zone-specific announcement (for Giant Isles zone ID)
    boss->GetMap()->SendZoneText(ZONE_GIANT_ISLES, message);
    
    // Server-wide announcement for major events
    ChatHandler(nullptr).SendWorldText(message);
}

// Announcement triggers:
// 1. Boss spawn: "[WORLD BOSS] Oondasta has awakened in the Giant Isles!"
// 2. 50% HP: "[WORLD BOSS] Oondasta roars in fury! (50% HP)"
// 3. 25% HP: "[WORLD BOSS] Oondasta is weakening! (25% HP)"
// 4. Death: "[WORLD BOSS] Oondasta has been slain by <Guild/Raid>!"
```

### Group Phasing for World Bosses

**Implementation Pattern (based on existing PhasedDuels system):**

```cpp
// World boss phasing - keeps boss fights isolated per raid group
class GiantIslesWorldBossScript : public CreatureScript
{
public:
    // Find available phase for raid group
    uint32 FindAvailablePhase(Creature* boss)
    {
        // Similar to DCPhasedDuels::FindFreePhase()
        // Start at phase 2 (phase 1 = normal world)
        for (uint32 phase = 2; phase <= 256; phase *= 2)
        {
            if (!IsPhaseInUse(phase))
                return phase;
        }
        return 1; // Fallback to normal phase
    }
    
    void OnCombatStart(Creature* boss, Unit* attacker)
    {
        Player* player = attacker->ToPlayer();
        if (!player)
            return;
        
        Group* group = player->GetGroup();
        if (!group || !group->isRaidGroup())
        {
            // Solo/party - no phasing, normal world boss
            return;
        }
        
        // Raid group - create phased instance
        uint32 phaseId = FindAvailablePhase(boss);
        
        // Move boss and all raid members to phase
        boss->SetPhaseMask(phaseId, true);
        
        for (GroupReference* itr = group->GetFirstMember(); itr != nullptr; itr = itr->next())
        {
            if (Player* member = itr->GetSource())
            {
                if (member->GetMap() == boss->GetMap())
                {
                    member->SetPhaseMask(phaseId, true);
                }
            }
        }
        
        // Mark phase as in use
        sActiveWorldBossPhases[boss->GetGUID()] = phaseId;
    }
    
    void OnCombatEnd(Creature* boss, bool killed)
    {
        // Return all participants to normal phase
        auto it = sActiveWorldBossPhases.find(boss->GetGUID());
        if (it != sActiveWorldBossPhases.end())
        {
            uint32 phaseId = it->second;
            
            // Reset all players in this phase to normal
            // (iterate map players in phase)
            
            boss->SetPhaseMask(PHASEMASK_NORMAL, true);
            sActiveWorldBossPhases.erase(it);
        }
    }
};
```

## ğŸ”„ Rotational Boss Spawn System (Hotspot-Style)

### Overview

Instead of fixed spawn times, world bosses use a **rotation system** similar to Hotspots:
- Only **1 world boss active at a time** (configurable)
- Bosses rotate through a spawn pool
- After one is killed, the next spawns after a delay
- Creates urgency and community coordination

### Configuration

```ini
# darkchaos-custom.conf

GiantIsles.RotationalBoss.Enable = 1

GiantIsles.RotationalBoss.AnnounceKill = 1       # Server-wide kill message
GiantIsles.RotationalBoss.AnnounceWarning = 1    # 10-min warning before spawn
```

### C++ Implementation (Hotspot-Inspired)

```cpp
// dc_giant_isles_rotational_boss.cpp

namespace GiantIslesRotation
{
    struct RotationalBossConfig
    {
        bool enabled = true;
        uint32 maxActive = 1;
        uint32 respawnDelayMinutes = 120;
        bool randomizeOrder = true;
        bool announceSpawn = true;
        bool announceKill = true;
        bool announceWarning = true;
        uint32 minPlayersOnline = 10;
    };
    
    static RotationalBossConfig sConfig;
    
    struct RotationalBoss
    {
        uint32 bossId;
        std::string bossName;
        uint32 creatureEntry;
        uint32 mapId;
        float x, y, z, o;
        uint32 spawnWeight;
        uint32 minPlayers;
        uint32 chaosAuraSpell;
        bool isEnabled;
        time_t lastKilled;
        uint32 totalKills;
    };
    
    static std::vector<RotationalBoss> sBossPool;
    static RotationalBoss* sCurrentBoss = nullptr;
    static ObjectGuid sCurrentBossGuid;
    static time_t sNextSpawnTime = 0;
    static bool sIsAlive = false;
    
    // Load boss pool from database
    void LoadBossPool()
    {
        sBossPool.clear();
        
        QueryResult result = WorldDatabase.Query(
            "SELECT boss_id, boss_name, creature_entry, spawn_map, spawn_x, spawn_y, spawn_z, spawn_o, "
            "spawn_weight, min_players, chaos_aura_spell, is_enabled, last_killed, total_kills "
            "FROM dc_giant_isles_rotational_bosses WHERE is_enabled = 1"
        );
        
        if (!result)
            return;
        
        do {
            Field* fields = result->Fetch();
            RotationalBoss boss;
            boss.bossId = fields[0].Get<uint32>();
            boss.bossName = fields[1].Get<std::string>();
            boss.creatureEntry = fields[2].Get<uint32>();
            boss.mapId = fields[3].Get<uint32>();
            boss.x = fields[4].Get<float>();
            boss.y = fields[5].Get<float>();
            boss.z = fields[6].Get<float>();
            boss.o = fields[7].Get<float>();
            boss.spawnWeight = fields[8].Get<uint32>();
            boss.minPlayers = fields[9].Get<uint32>();
            boss.chaosAuraSpell = fields[10].Get<uint32>();
            boss.isEnabled = fields[11].Get<bool>();
            boss.lastKilled = fields[12].IsNull() ? 0 : fields[12].Get<int64>();
            boss.totalKills = fields[13].Get<uint32>();
            
            sBossPool.push_back(boss);
        } while (result->NextRow());
        
        LOG_INFO("scripts", "Giant Isles: Loaded {} rotational bosses", sBossPool.size());
    }
    
    // Select next boss (weighted random)
    RotationalBoss* SelectNextBoss()
    {
        if (sBossPool.empty())
            return nullptr;
        
        if (!sConfig.randomizeOrder)
        {
            // Sequential: find boss with oldest lastKilled
            RotationalBoss* oldest = &sBossPool[0];
            for (auto& boss : sBossPool)
            {
                if (boss.lastKilled < oldest->lastKilled)
                    oldest = &boss;
            }
            return oldest;
        }
        
        // Weighted random selection
        uint32 totalWeight = 0;
        for (auto& boss : sBossPool)
            totalWeight += boss.spawnWeight;
        
        uint32 roll = urand(0, totalWeight - 1);
        uint32 cumulative = 0;
        
        for (auto& boss : sBossPool)
        {
            cumulative += boss.spawnWeight;
            if (roll < cumulative)
                return &boss;
        }
        
        return &sBossPool[0];
    }
    
    // Spawn the current boss
    void SpawnBoss(RotationalBoss* boss)
    {
        if (!boss)
            return;
        
        Map* map = sMapMgr->CreateBaseMap(boss->mapId);
        if (!map)
            return;
        
        // Summon the creature
        if (Creature* creature = map->SummonCreature(boss->creatureEntry, 
            {boss->x, boss->y, boss->z, boss->o}))
        {
            sCurrentBossGuid = creature->GetGUID();
            sIsAlive = true;
            sCurrentBoss = boss;
            
            // Apply chaos corruption aura
            if (boss->chaosAuraSpell)
                creature->CastSpell(creature, boss->chaosAuraSpell, true);
            
            // Server-wide announcement
            if (sConfig.announceSpawn)
            {
                std::string msg = Acore::StringFormat(
                    "|cffff0000[WORLD BOSS]|r |cff00ff00%s|r has awakened in the Giant Isles!",
                    boss->bossName.c_str()
                );
                ChatHandler(nullptr).SendWorldText(msg.c_str());
                
                // Zone announcement
                map->SendZoneText(ZONE_GIANT_ISLES, msg.c_str());
            }
            
            LOG_INFO("scripts", "Giant Isles: Spawned rotational boss {} (entry {})",
                     boss->bossName, boss->creatureEntry);
        }
    }
    
    // Handle boss death
    void OnBossDeath(Creature* boss, Unit* killer)
    {
        if (!sCurrentBoss)
            return;
        
        sIsAlive = false;
        sCurrentBoss->lastKilled = GameTime::GetGameTime().count();
        sCurrentBoss->totalKills++;
        
        // Update database
        WorldDatabase.Execute(
            "UPDATE dc_giant_isles_rotational_bosses SET last_killed = NOW(), total_kills = {} WHERE boss_id = {}",
            sCurrentBoss->totalKills, sCurrentBoss->bossId
        );
        
        // Schedule next spawn
        sNextSpawnTime = GameTime::GetGameTime().count() + (sConfig.respawnDelayMinutes * 60);
        
        // Announce
        if (sConfig.announceKill)
        {
            std::string killerName = "adventurers";
            if (Player* player = killer->ToPlayer())
            {
                if (Guild* guild = player->GetGuild())
                    killerName = guild->GetName();
            }
            
            std::string msg = Acore::StringFormat(
                "|cff00ff00[WORLD BOSS]|r |cffff0000%s|r has been slain by |cfffff000%s|r! "
                "Next boss spawns in |cff00ffff%u|r minutes.",
                sCurrentBoss->bossName.c_str(), killerName.c_str(), sConfig.respawnDelayMinutes
            );
            ChatHandler(nullptr).SendWorldText(msg.c_str());
        }
        
        sCurrentBoss = nullptr;
        sCurrentBossGuid.Clear();
    }
    
    // Update tick (called every minute via WorldScript)
    void Update()
    {
        if (!sConfig.enabled || sIsAlive)
            return;
        
        time_t now = GameTime::GetGameTime().count();
        
        // Check if it's time to spawn
        if (sNextSpawnTime > 0 && now >= sNextSpawnTime)
        {
            // Check player count
            uint32 onlinePlayers = sWorld->GetPlayerCount();
            
            RotationalBoss* nextBoss = SelectNextBoss();
            if (nextBoss && onlinePlayers >= nextBoss->minPlayers)
            {
                SpawnBoss(nextBoss);
                sNextSpawnTime = 0;
            }
            else if (sConfig.announceWarning && nextBoss)
            {
                // Not enough players, announce waiting
                std::string msg = Acore::StringFormat(
                    "|cffff8800[WORLD BOSS]|r %s is ready to spawn but requires %u players online (current: %u)",
                    nextBoss->bossName.c_str(), nextBoss->minPlayers, onlinePlayers
                );
                ChatHandler(nullptr).SendWorldText(msg.c_str());
            }
        }
        
        // 10-minute warning
        if (sConfig.announceWarning && sNextSpawnTime > 0)
        {
            time_t timeUntilSpawn = sNextSpawnTime - now;
            if (timeUntilSpawn > 0 && timeUntilSpawn <= 600 && timeUntilSpawn > 540)
            {
                RotationalBoss* nextBoss = SelectNextBoss();
                if (nextBoss)
                {
                    std::string msg = Acore::StringFormat(
                        "|cffff8800[WORLD BOSS]|r %s will spawn in approximately 10 minutes!",
                        nextBoss->bossName.c_str()
                    );
                    ChatHandler(nullptr).SendWorldText(msg.c_str());
                }
            }
        }
    }
}
```
