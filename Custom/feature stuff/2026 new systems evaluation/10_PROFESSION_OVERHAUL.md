# Profession Overhaul

**Priority:** B2 (Medium)  
**Effort:** High (4-6 weeks)  
**Impact:** High  
**Client Required:** No

---

## Overview

Make professions relevant at endgame (160-255). New recipes, unique perks, progression path.

---

## Problem Statement

Current state at level 255:
- Professions capped at 450
- Recipes are trivial
- No endgame crafting value
- Gathering is pointless

---

## Proposed Changes

### Profession Extension to 600

| Skill Range | Content |
|-------------|---------|
| 450-525 | New rare materials |
| 525-575 | Epic crafted gear |
| 575-600 | Legendary patterns |

### New Material Tiers

| Material | Source | Skill Required |
|----------|--------|----------------|
| Chaos Ore | Mining 160+ zones | 475 |
| Void Leather | Skinning void creatures | 475 |
| Twilight Herb | Herbalism endgame zones | 475 |
| Primordial Cloth | Tailoring via quests | 500 |

### Crafted Gear Value

| Item Tier | Equivalent To | Upgrade Potential |
|-----------|---------------|-------------------|
| Rare (525) | M+5 loot | Can upgrade to T1 |
| Epic (575) | M+10 loot | Can upgrade to T2 |
| Legendary (600) | M+15 loot | Starts at T2 |

### Profession Perks (Endgame)

| Profession | Perk at 600 |
|------------|-------------|
| Blacksmithing | +5% damage with crafted weapons |
| Leatherworking | +5% dodge/armor |
| Tailoring | +10% mana pool |
| Alchemy | Double flask duration |
| Enchanting | Ring enchants (no slot restriction) |
| Jewelcrafting | Unique meta gems |
| Engineering | Combat gadgets |
| Inscription | Combat glyphs |

---

## Implementation

### Phase 1 (Week 1-2)
- Extend skill caps
- Add new trainers
- Create material items

### Phase 2 (Week 3-4)
- Recipe designs
- Crafting costs
- Integration with upgrade system

### Phase 3 (Week 5-6)
- Profession perks
- Daily cooldowns
- Questlines for patterns
